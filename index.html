How It Works

Four components working in perfect sync

01

Indexer

Continuously downloads trades for all mints into a massive table. Real-time data ingestion from the entire Solana ecosystem.

→
02

Featurizer

Calculates features for ML models: trading volumes, volatility, transaction intensity, deal parameters, and more.

→
03

ML Scoring

ML model outputs a score for each mint. Higher score = higher confidence in price growth. Real-time predictions.

→
04

Executor

Executes trades with optimal parameters: routing, slippage, fees. Smart order management.

End-to-End Analytics

All model parameters, bot configurations, and trade data flow into ClickHouse and are visualized in Grafana. We use this data for experiment analysis, discovering new opportunities, and expanding into new markets.
